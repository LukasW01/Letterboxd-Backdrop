# Letterboxd-Backdrop
Browser extension that adds a custom backdrop image to your Letterboxd profile without paying for Patron.

![letterboxd-profile](src/files/letterboxd-profile.png)

###  Getting Started

Install packages:

```bash
pnpm i 
```

Build the extension:

```bash
pnpm build
```
